difference between git and GitHub

| Feature       | Git 🧠             | GitHub 🌐          |
| ------------- | ------------------ | ------------------ |
| Type          | Tool / Software    | Platform / Service |
| Usage         | Track code changes | Host & share code  |
| Internet      | Not required       | Required           |
| Collaboration | Limited (local)    | Strong (teams)     |
| Installed?    | Yes                | No (browser-based) |
